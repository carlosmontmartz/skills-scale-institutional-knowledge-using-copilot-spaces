# OctoAcme Project Management Process Docs

This README collects and links all project management process documentation for OctoAcme. It helps teams find guidance quickly and ensures a shared understanding of our approach.

## Overview of OctoAcme Project Management Processes

- **Customer-first, iterative delivery:** Focus on clear ownership and measurable results.
- **Lifecycle stages:** Initiation, Planning, Execution, Release, Retrospective & Continuous Improvement
- **Clear roles:** Project Manager, Product Manager, Developer, QA, Stakeholder
- **Key artifacts:** Project Charter/One-pager, Backlog, Acceptance Criteria, Risk Register, Release notes, Retrospectives
- **Regular communication:** Weekly syncs, standups, and updates to stakeholders
